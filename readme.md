# Cart-Pole Collocation

A project for solving the cart-pole control problem using collocation methods.

### Installation

```bash
git clone https://github.com/FluffyCodeMonster/cart-pole-collocation
cd cart-pole-collocation
pip install -r requirements.txt
```
